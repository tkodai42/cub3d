## USAGE
```
make
./cub3D       < map path >

make bonus
./cub3D       < map path >
```
## Controls(bonus)
 
| Key | Action |
|---|---|
| <kbd>W</kbd>, <kbd>A</kbd>, <kbd>S</kbd>, <kbd>D</kbd> | MOVE |
| <kbd>E</kbd>| interact |
| <kbd>←</kbd>, <kbd>→</kbd>| TURN |
| <kbd>SP</kbd>| JUMP |
| <kbd>shift</kbd>| CROUCH |
| <kbd>↓</kbd>| SPRINT |
| <kbd>↑</kbd>| SHOT |
| <kbd>M</kbd>| display map |
| <kbd>1</kbd>, <kbd>2</kbd>, <kbd>3</kbd> | weapons |
| <kbd>4</kbd>| melee |
| <kbd>ESC</kbd>| EXIT |
